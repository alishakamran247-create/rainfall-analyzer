# 🌧️ Rainfall Statistics Analyzer

A console-based C++ program that analyzes monthly rainfall data for two consecutive years and generates statistical insights. This was my lab project at the University of Central Punjab.

## 🚀 Features

- 📥 Input rainfall data for 12 months (current & previous year)
- 📊 Display a formatted monthly rainfall table
- 🔢 Calculate **total rainfall** for both years
- 📈 Calculate **average monthly rainfall** for both years
- 🏔️ Find the **highest rainfall month** for both years
- ☀️ Identify **months with no rainfall**
- 📉 Generate an **ASCII bar chart** of rainfall data

## 🛠️ Technologies Used

- **Language:** C++
- **Concepts:** Arrays, Loops, Functions, Console Output Formatting

## 📁 How to Run

1. Open the project in any C++ IDE (Dev-C++, VS Code, Code::Blocks)
2. Compile `lab_project.cpp`
3. Run the executable
4. Enter rainfall values for each month when prompted (current year then previous year)

## 💡 Sample Input

```
Month 1: 12.5 10.0
Month 2: 0 5.5
Month 3: 8.3 7.1
...
```
